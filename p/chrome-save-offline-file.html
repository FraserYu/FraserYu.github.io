Chrome 浏览器保存网页为脱机文件

我们经常需要将一些网页保存为脱机文件，这样即便没有网络我们也可以正常的浏览，IE浏览器将网页另存为在选择文件类型的地方会有选择*.mht, 可是Chrome 却没有这个选项，接下来教大家如何通过Chrome来保存网页为mht文件

  1. 打开浏览器输入 chrome://flags/

  2. 通过关键字搜索save 或 另存为如下图：

  3. 点击Enable或使用，重新启动浏览器

  4. 鼠标右键另存为选择类型和保存路径如下图：

  5. 等待下载完成即可
